# 災情回報檢查表

## 概述
確保災情回報的完整性與準確性，供資訊回報 Agent 參考。

## 檢查項目

- [ ] **地理位置確認**：
  - [ ] 經緯度是否精確？
  - [ ] 詳細地址或地標描述是否清晰？
  - [ ] 地點補充說明是否足夠？

- [ ] **災情類型與描述**：
  - [ ] 災情類型是否正確勾選？
  - [ ] 災情詳細描述是否具體、客觀？
  - [ ] 是否有補充說明？

- [ ] **緊急程度評估**：
  - [ ] 緊急程度（極緊急/緊急/一般）是否符合現場實際情況？

- [ ] **傷亡情況確認**：
  - [ ] 受傷、死亡、失蹤、受困人數是否填寫？
  - [ ] 傷亡情況補充說明是否清晰？

- [ ] **基礎設施損壞情況**：
  - [ ] 基礎設施類型是否勾選？
  - [ ] 損壞狀態是否明確？
  - [ ] 是否有補充說明？

- [ ] **現場需求確認**：
  - [ ] 需求類型是否勾選？
  - [ ] 需求補充說明是否具體？

- [ ] **媒體附件**：
  - [ ] 是否上傳了照片/影片連結？
  - [ ] 照片/影片描述是否清晰？
